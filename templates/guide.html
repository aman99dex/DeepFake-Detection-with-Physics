↓ Data Preparation
1
Download Free Datasets (Auto)
FREE — Auto Download

This downloads ~120K real/fake face images from HuggingFace and organizes them into a unified training set. No sign-up required.

# Install HuggingFace hub first (if not installed) pip install huggingface_hub datasets # Download and prepare everything python scripts/prepare_data.py # Check what was downloaded python scripts/prepare_data.py --check

This gives you ~120K images — enough for a good first training run.

2
FaceForensics++ (Academic Benchmark)
Manual — Free for Research

The gold-standard deepfake benchmark. 1,000 original + 4,000 manipulated videos. Required for publishing results.

# 1. Request access (fill a short form): # https://github.com/ondyari/FaceForensics # 2. Download and extract video frames # 3. Place extracted frames at: # data/raw/ff++/real/ ← frames from original_sequences/ # data/raw/ff++/fake/ ← frames from any manipulation (Deepfakes, Face2Face, etc.) # 4. Rebuild unified dataset including FF++ python scripts/prepare_data.py --include-manual
3
CelebDF-v2
Manual — Free Direct Download

590 real + 5,639 synthesized celebrity videos. High visual quality, harder to detect. No sign-up needed.

# Download from GitHub: # https://github.com/yuezunli/celeb-deepfakeforensics # Place frames at: # data/raw/celeb-df/real/ # data/raw/celeb-df/fake/ python scripts/prepare_data.py --include-manual
4
DFDC (DeepFake Detection Challenge)
Manual — Kaggle Account Required

The largest deepfake dataset: 100K+ video clips, multiple generators. Most challenging real-world benchmark.

# Requires free Kaggle account # https://kaggle.com/competitions/deepfake-detection-challenge/data # Place frames at: # data/raw/dfdc/real/ # data/raw/dfdc/fake/
▶ Training
5
Train v2 Extended (10-score model)
Recommended

The new train_v2.py script trains the 10-score model with mixed precision, gradient accumulation, and warmup scheduling.

# Quick test (5 epochs, makes sure everything works) python train_v2.py --epochs 5 --max-samples 200 --synthetic-fallback # Standard run with free HuggingFace data python train_v2.py --data data/processed/unified --epochs 30 # Full quality training (needs GPU, 50 epochs) python train_v2.py --data data/processed/unified --epochs 50 \ --nerf-hidden 256 --nerf-layers 8 --sample-points 1024 \ --batch-size 16 --gtr-samples 128 # Resume from checkpoint python train_v2.py --resume outputs/v2_extended/checkpoints/last.pt
GPU Tip: Training on CPU is possible but slow. For 30 epochs on ~100K images, expect ~6-8 hours on CPU vs ~45 minutes on a T4 GPU (free on Google Colab).
6
Cross-Dataset Evaluation
Critical Test

Train on one dataset, test on others without retraining. This is the real test of physics-based generalization.

# Evaluate best model across all available datasets python scripts/cross_dataset_eval.py --model v2_extended # Compare v1 vs v2_extended python scripts/cross_dataset_eval.py \ --checkpoints outputs/real_data/checkpoints/best_model.pt \ outputs/v2_extended/checkpoints/best_model.pt \ --names V1 V2_Extended --model v2_extended
Expected: AUC drops ~2-5% across datasets for PhysForensics (physics generalizes). Compare with XceptionNet which drops from 0.997 on FF++ to ~0.65 on CelebDF.
